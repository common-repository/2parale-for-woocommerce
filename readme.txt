== Description ==
An integration plugin for 2Parale's tracking code in WooCommerce based sites

== Installation ==
Make sure you have WooCommerce installed

1. Extract the woo-2parale folder to your wp-content/plugins directory
2. Activate the plugin in the admin interface
3. Go to /wp-admin/admin.php?page=wc-settings&tab=integration (WooCommerce > Settings > Integration > 2Parale Sale Tracking)
4. Fill in your campaign unique and confirmation code (you can find these in the 2Parale interface, in the tracking code setup guide)

That's it!
